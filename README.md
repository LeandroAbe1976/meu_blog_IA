# 🚀 Meu Blog Pessoal - Atividade TADS

Olá! Este é o repositório do meu primeiro blog, desenvolvido para a faculdade.

## 🤖 IAs Utilizadas
Para este projeto, utilizei a IA **Gemini (do Google)** para:
- Estruturar o código HTML e CSS.
- Explicar conceitos de desenvolvimento de forma didática.
- Organizar este arquivo README.

## 🛠️ Tecnologias
- **HTML5:** Estrutura do conteúdo.
- **CSS3:** Estilização e cores.
- **Git/GitHub:** Controle de versão e hospedagem.

## 📚 O que aprendi
Como estudante de TADS, aprendi a importância de separar o conteúdo (HTML) do visual (CSS) e como o GitHub é essencial para salvar e compartilhar códigos com o mundo.
